🎁 Downloading MGnify Analyses for a given Study#

In this notebook, is starting code for retrieving all the MGnify Analyses for a given study accession. We use MGYS00010442 as an example.

🎯 The Goal: View MGnify Analyses for study: MGYS00010442 #

In this tutorial we demonstrate how MGni.py can be used to get MGnify Analyses datasets for a study.

%load_ext autoreload
%autoreload 2

# uncomment below if colab
#!pip install mgnipy
#!pip install asyncio

# our study of interset
study_accession = "MGYS00010442"

🔎 Getting the Study’s details#

Here we import and configuire our MGnipy client

from mgnipy import MGnipy
import asyncio # optional, for async requests

# init
MG = MGnipy(
    #cache_dir="cache" # default dir is platform-specific,set to None to disable caching
)

# access study resource
study = MG.study(study_accession)

# check out call before making it
study.dry_run()

# get

📒 MGazine for accessing the study’s datasets#

  • study details have a mgnipy.MGazine which allow us to download and interact with study-level datasets outputed from MGnify.

  • We can use mgnipy.MGazine to download the datasets onto disk or read them into our notebook.

  • To access the study’s mgazine use .datasets

  • the str representaiton of mgazine tells us the number of download files and their {alias: url}

# getting mgazine
mgazine = study.datasets
# see info
print(mgazine)
